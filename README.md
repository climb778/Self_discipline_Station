# 自律小站

一款面向个人的自律打卡与任务管理应用，帮助你规划每日待办、专注执行、复盘成长。基于 uni-app + Vue 3 开发，数据全部存储在本地，支持 H5 和微信小程序双端运行。

## 技术栈

- **框架**: uni-app + Vue 3 Composition API (`<script setup>`)
- **构建**: Vite 5.2.8
- **存储**: `uni.getStorageSync` / `uni.setStorageSync`（纯本地存储，无后端）
- **平台**: H5、微信小程序（mp-weixin）

## 运行方式

```bash
# 安装依赖
npm install

# H5 开发（浏览器访问 http://localhost:5173）
npm run dev:h5

# 微信小程序开发（编译后用微信开发者工具打开 dist/dev/mp-weixin）
npm run dev:mp-weixin

# H5 生产构建
npm run build:h5

# 微信小程序生产构建
npm run build:mp-weixin
```

## Android APK 打包

本项目支持通过 HBuilderX 打包为 Android APK，安装到手机长期自用。

### 打包步骤

1. 使用 HBuilderX 打开项目根目录
2. 菜单栏选择 **发行 → 原生 App-云打包**（或本地打包）
3. 选择 Android 平台，填写打包配置：
   - AppID：首次云打包时在 HBuilderX 中获取/绑定正式 AppID
   - 包名：默认可使用 `com.selfdiscipline.station`
   - 证书：使用调试证书或自定义证书
4. 点击打包，等待完成后下载 APK 文件
5. 将 APK 传到手机安装即可

### 注意事项

- 应用已配置为竖屏模式，适合手机使用
- Android 打包权限已收敛为自定义空权限，如需相册/文件能力再按需开启
- 所有数据存储在本地，卸载后数据会丢失，建议定期通过设置页导出备份
- 最低支持 Android 5.0（API 21）
