# 自律小站

一款面向个人的自律打卡与任务管理应用，帮助你规划每日待办、专注执行、复盘成长。基于 uni-app + Vue 3 开发，数据全部存储在本地，支持 H5 和微信小程序双端运行。

## 技术栈

- **框架**: uni-app + Vue 3 Composition API (`<script setup>`)
- **构建**: Vite 5.2.8
- **存储**: `uni.getStorageSync` / `uni.setStorageSync`（纯本地存储，无后端）
- **平台**: H5、微信小程序（mp-weixin）

## 核心功能

| 模块 | 功能 |
|------|------|
| 每日待办 | 每日任务列表、完成率统计、待办提醒卡片 |
| 岁月打卡 | 日历视图、按月查看完成状态、每日任务详情 |
| 过往记录 | 搜索筛选、已完成/长期/过期/归档分类 |
| 我的主页 | 用户资料、行动仪表盘、7天趋势图、快捷入口 |
| 任务详情 | 查看全部字段、标记完成、编辑、归档、删除 |
| 重复任务 | 每天/每周/每月/自定义星期、自动生成功能 |
| 应用内提醒 | 任务提醒卡片、最近提醒时间展示 |
| 任务模板 | 按分类快速创建预设任务 |
| 设置中心 | 个人信息、主题换肤、提醒/专注设置、数据管理 |
| 数据导入导出 | JSON 导出/导入、演示数据生成、清空数据 |
| 主题换肤 | 默认蓝、森林绿、暖橙色三套主题 |
| 成就徽章 | 7项成就：首次完成、连续打卡、效率大师等 |
| 任务归档 | 归档不删除，过往记录中可查 |
| 番茄钟专注模式 | 15/25/45分钟或自定义时长、暂停/继续/结束 |
| 专注记录 | 专注统计、按任务查看、今日/本周/累计 |
| 周报/月报数据报告 | 核心数据、7天趋势图、五分类统计、智能总结、报告卡片 |

## 运行方式

```bash
# 安装依赖
npm install

# H5 开发（浏览器访问 http://localhost:5173）
npm run dev:h5

# 微信小程序开发（编译后用微信开发者工具打开 dist/dev/mp-weixin）
npm run dev:mp-weixin

# H5 生产构建
npm run build:h5

# 微信小程序生产构建
npm run build:mp-weixin
```

所有数据存储在浏览器或小程序的本地存储中，不会上传到任何服务器。可通过设置页的「导出 JSON」功能备份数据，也可通过「导入 JSON」恢复。
