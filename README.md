# 自律小站

个人自律打卡和每日任务管理应用，基于 uni-app + Vue 3 开发，支持 H5 和微信小程序。

## 功能概览

### 任务管理
- 新增/编辑/删除/归档任务
- 任务分类（学习、生活、工作、运动、其他）
- 优先级标记（高/中/低）
- 长期任务支持
- 重复任务（每天/每周/每月/自定义星期）
- 任务提醒

### 每日待办
- 每日任务列表，完成率实时统计
- 待办提醒卡片，显示未完成、过期、最近提醒

### 专注模式
- 番茄钟计时器，预设 15/25/45 分钟或自定义时长
- 支持暂停、继续、提前结束
- 完成弹窗：标记任务完成 / 只保存记录 / 继续专注
- 专注记录持久化，支持数据导出/导入

### 岁月打卡
- 日历视图，按月查看每日任务完成状态
- 每日任务详情和统计

### 过往记录
- 搜索和筛选已完成/长期/过期/归档任务
- 任务完成时间线

### 数据报告
- 本周/本月报告切换
- 核心数据：完成任务、新增任务、过期任务、完成率、专注分钟、专注次数
- 7 天任务完成趋势柱状图
- 7 天专注时长趋势柱状图
- 五分类统计（学习/生活/工作/运动/其他）含占比和进度条
- 智能总结文案自动生成
- 报告卡片适合截图分享

### 成就系统
- 7 项成就：首次完成、连续打卡 3/7 天、效率大师、百项计划、学习之星、运动达人

### 个人中心
- 用户资料（昵称、签名、头像）
- 行动仪表盘（完成率、连续打卡、本周/月完成、最长连续、最常用分类）
- 7 天完成趋势图
- 快捷入口（新增任务、今日待办、开始专注、数据报告、过往记录）

### 设置
- 主题换肤（默认蓝、森林绿、暖橙色）
- 提醒设置
- 专注设置（默认时长、提示音开关）
- 数据导出/导入/清空/生成演示数据

## 技术栈

- **框架**: uni-app + Vue 3 Composition API
- **构建**: Vite 5.2.8
- **存储**: uni.getStorageSync / uni.setStorageSync（本地存储）
- **平台**: H5、微信小程序

## 快速开始

```bash
# 安装依赖
npm install

# H5 开发
npm run dev:h5

# 微信小程序开发
npm run dev:mp-weixin

# H5 构建
npm run build:h5

# 微信小程序构建
npm run build:mp-weixin
```

## 数据说明

所有数据存储在浏览器/小程序本地存储中，不会上传到任何服务器。可通过设置页的"导出 JSON"功能备份数据。
